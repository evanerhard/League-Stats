Welcome to League-Stats!

Fantasy Football requires competitors to be up-to-date with new and current player statistics. League-Stats works with real-time data to keep the subscribers up to date with the newest statistics posted by the NFL. League-Stats allows users to visually search for real-time NFL player statistics, view data on a weekly graph, and compare players in a simple yet detailed format. League-Stats scrapes the NFL players stats from the official NFL website and stores the data within a database using PostgreSQL, and is graphed using Chart.js.
